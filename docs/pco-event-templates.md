# Planning Center Event Templates

Planning Center Event Templates help staff create consistently tagged and
roomed events. They are optional, but useful when the same event types recur.

Templates are different from public request forms. Templates help staff create
events directly in Planning Center; forms let requesters submit events for
approval.

## Suggested Templates

| Template | Tags | Rooms/Resources | Comms effect |
| --- | --- | --- | --- |
| Weekend Service | `Service`, `All Church` | Main auditorium | Tier 1 |
| Funeral or Memorial | `Funeral` | Chapel, reception room | No-promo |
| Wedding | `Wedding` | Auditorium or chapel, reception room | No-promo |
| Small Group | `Small Group` | Meeting room | Tier 3 |
| Class or Study | `Class`, ministry tag | Classroom | Tier from ministry |
| Staff Meeting | `Meeting`, `Staff` | Meeting room | No-promo |
| Fellowship Event | `Meeting` or `Room Only` | Fellowship room | No-promo unless triaged |
| Kids Event | `Kids` or `Children's`, `Class` | Kids rooms | Tier 2 |
| Student Event | `Students` or `Youth` | Student room | Tier 2 |
| Mission Trip | `All Church`, `Mission Trip` | Optional info meeting room | Tier 1 and playbook |

## Build Flow

In Planning Center Calendar:

1. Create or confirm the tag groups from `docs/pco-forms-setup.md`.
2. Create one template at a time.
3. Add a clear name and public description.
4. Add the preset tags.
5. Add standard rooms/resources.
6. Save and test by creating one event from the template.
7. Import the approved event into Church Comms and confirm classification.

Keep template names, tags, and room names aligned with the rules in Church
Comms Settings.
